About The Project

Inbox AI is an intelligent tool designed to streamline email management. It functions as a Chrome extension that can instantly generate professional, casual, or friendly replies to any email. The goal is to save users time and improve the quality of their communications. The front-end is built with React, and the powerful AI backend is powered by Spring Boot.

Technologies Used

  • React (Front-end & Extension UI)
  • Spring Boot (Back-end)
  • Chrome Extension APIs
  • AI/ML Model Integration (for email generation)
