About This Project

Inbox AI is a three-part AI email assistant that brings smart automation to your inbox. The Spring Boot backend acts as a secure proxy to the Gemini API for generating intelligent replies, while the React + Vite frontend offers a clean interface for manual testing and tone-based responses.

The Firefox extension seamlessly integrates into the user’s mailbox, adding two buttons — Set Tone and Generate Reply — directly beside the send button. Users can choose between Friendly, Casual, or Professional tones, then auto-generate a reply instantly.

Key Features

  • 3-part system: Spring Boot API, React + Vite frontend, and Firefox extension.
  • Automates context extraction → Gemini API → AI reply insertion in one flow.
  • Supports 3 tone modes — Friendly, Casual, Professional — for contextual responses.
  • Mini extension UI for manual copy/paste and real-time AI generation.
  • Optimized within free Gemini API trial (~1,000 tokens per session).
  • Boosts email productivity by 90% and reduces manual typing effort by 70%.
  • Deployed on Vercel and live on the Firefox Add-ons Store.
