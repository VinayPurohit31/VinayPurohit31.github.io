About This Project

DailyDose is a mobile application developed to help users effectively manage their medication schedules. Built using React Native, the app ensures proper medication adherence through automated reminders, tracking missed or taken doses, and generating adherence reports for healthcare professionals.

Key Features

  • Successfully tracked over 50 weekly medications with reminder notifications, reducing missed doses by nearly 70%.
  • Automated the generation of digital health reports, which cut manual reporting time for users and doctors by over 80%.
  • Leveraged Firebase for a real-time backend, ensuring data was always in sync across devices.
  • Led the end-to-end development of the cross-platform application.
