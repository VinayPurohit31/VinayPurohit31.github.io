About This Project

Messify is a native Android application designed to help students and professionals quickly find local food mess services. The app provides a convenient way to discover, view details, and contact services within a 5-kilometer radius, significantly improving convenience for users looking for daily meal options.

Key Features

  • Designed and built over 15 interactive and user-friendly application screens using modern Android UI principles.
  • The core feature boosted user convenience by an estimated 60% based on initial user feedback.
  • Developed entirely with native technologies (Kotlin and XML) for optimal performance and a seamless user experience on Android devices.
